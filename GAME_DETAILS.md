# Name That Color - Game Details

## Game Mechanics

### 20 Color Rounds
The game features 20 different color swatches, each with a unique official color name:

1. **Scarlet** - Deep red (#DC143C)
2. **Chartreuse** - Yellow-green (#7FFF00)
3. **Cerulean** - Deep blue (#4169E1)
4. **Vermillion** - Bright red-orange (#FF3F00)
5. **Taupe** - Brownish-gray (#B38B6D)
6. **Magenta** - Purple-pink (#FF00FF)
7. **Periwinkle** - Light blue-purple (#CCFFFF)
8. **Celadon** - Light gray-green (#ACE1AF)
9. **Turquoise** - Blue-green (#40E0D0)
10. **Lavender** - Light purple (#E6E6FA)
11. **Indigo** - Deep purple-blue (#4B0082)
12. **Cyan** - Bright blue-green (#00FFFF)
13. **Sienna** - Red-brown (#A0522D)
14. **Ochre** - Golden brown (#CC7000)
15. **Maroon** - Dark red (#800000)
16. **Jade** - Green (#00A86B)
17. **Coral** - Orange-pink (#FF7F50)
18. **Salmon** - Light orange-pink (#FA8072)
19. **Navy** - Dark blue (#000080)
20. **Plum** - Purple (#DDA0DD)

### Scoring System
- **Correct Answer**: +10 points
- **Wrong Answer**: -1 point
- **Final Score Display**: Shows total score earned
- **Accuracy Percentage**: Calculated as (correct / 20) × 100

### User Interface Features
- **Progress Bar**: Visual indication of progress through 20 rounds
- **Round Counter**: Shows current round number and total rounds
- **Live Score Display**: Updates as you answer each question
- **Color Swatch**: Large, clear display of the color to identify
- **Multiple Choice**: 4 options per question, shuffled randomly
- **Instant Feedback**: 
  - Green highlight for correct answer
  - Red highlight for incorrect answer
  - Shows the correct answer if you get it wrong
- **Final Statistics Screen**:
  - Total score
  - Number of correct answers (out of 20)
  - Number of wrong answers (out of 20)
  - Accuracy percentage
  - Personalized message based on performance

### Responsive Design
- Works on desktop, tablet, and mobile devices
- Flexible grid layout for option buttons
- Touch-friendly button sizing
- Responsive color swatch sizing

### Performance Indicators
- **Trophy (🏆)**: 80% or higher accuracy
- **Celebration (🎉)**: 60-79% accuracy
- **Thumbs Up (👍)**: 40-59% accuracy
- **Flexing Muscle (💪)**: Below 40% accuracy
