# Name That Color

An educational web game where players test their knowledge of color terminology by guessing the official names of 20 different color swatches.

## Game Features

- **20 color rounds** with unique hues ranging from common (red, blue) to obscure (chartreuse, vermillion, celadon)
- **Multiple choice** format with 4 options per round
- **Score tracking**: +10 for correct, -1 for wrong
- **Progress bar** to track your position in the game
- **Final statistics**: Score, accuracy percentage, correct/wrong ratio
- **Responsive design** works on desktop and mobile
- **No time pressure** - take your time to make your choice

## Colors Included

Scarlet, Chartreuse, Cerulean, Vermillion, Taupe, Magenta, Periwinkle, Celadon, Turquoise, Lavender, Indigo, Cyan, Sienna, Ochre, Maroon, Jade, Coral, Salmon, Navy, Plum

## Technologies

- **HTML5** - Semantic markup
- **CSS3** - Responsive design with gradients and animations
- **Vanilla JavaScript** - No frameworks or dependencies

## How to Play

1. Click "Start Game" to begin
2. Look at the colored swatch
3. Choose what you think is the correct color name from the 4 options
4. Get instant feedback on your answer
5. Progress through all 20 rounds
6. See your final score and accuracy

## Installation

Simply open `index.html` in any modern web browser. No build process or dependencies required.
